 TriSpace - Property & Service Management Platform

A full-stack web application built with React and Node.js for managing properties and tiffin (meal delivery) services.
This platform enables users to browse properties, post new listings, and access meal delivery services all in one place.

## 🚀 Features

- **User Authentication**: Secure login and registration system with JWT-based authentication
- **Property Management**: Browse, search, and filter properties with detailed information
- **Property Listings**: Users can post and manage their property listings
- **Admin Dashboard**: Comprehensive admin panel for managing properties and users
- **Responsive Design**: Mobile-friendly interface built with modern React components
- **Real-time Updates**: Dynamic property cards and live filtering capabilities
- **Secure Data Handling**: Password encryption and secure API endpoints

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI library
- **Vite** - Build tool and development server
- **Axios** - HTTP client
- **React Router** - Client-side routing
- **Context API** - State management
- **CSS3** - Styling

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB ODM
- **JWT** - Authentication
- **Bcrypt** - Password hashing

## 📋 Prerequisites

Before running this project, make sure you have:
- Node.js (v14 or higher)
- npm or yarn package manager
- MongoDB (local or Atlas connection string)
- Git
